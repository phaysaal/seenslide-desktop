# SeenSlide Icons

## Logo File

Place the SeenSlide logo file here:

**File name:** `seenslide_logo.png`

**Location:** `gui/resources/icons/seenslide_logo.png`

The logo should be the cyan-blue-purple gradient arrows forming an "S" shape (Image #5 from the design).

This logo will be used throughout the GUI application:
- Mode selector window
- Direct Talk window
- Conference launcher
- System tray icon

## Recommended Specifications

- Format: PNG with transparency
- Size: At least 512x512 pixels (will be scaled as needed)
- Color: Cyan-blue-purple gradient as shown in the original design
